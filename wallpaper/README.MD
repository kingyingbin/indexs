自采集壁纸网站源码
========
特色：壁纸种类多、自动更新，自动采集、支持多分辨率、瀑布流加载、全站 ajax!

壁纸采集自 360壁纸库、必应首页的每日图片以及金山词霸开放平台

相关原理见 [http://mkblog.cn/581/](http://mkblog.cn/581/)  和  [http://mkblog.cn/492/](http://mkblog.cn/492/)

本作品禁止任何形式的倒卖，转载请注明出处！
